Introspected Table Attributes

After the database introspection step, and before code generation begins, Ibator calculates many attributes that are used by the actual code generators. Plugins can override any of these attributes before code generation by implementing the initialized method.

Attribute Details
ATTR_BASE_RECORD_TYPE
Data Type: org.apache.ibatis.ibator.api.dom.java.FullyQualifiedJavaType
Setter: setBaseRecordType
Getter: getBaseRecordType
Description: The fully qualified type of the base record. By default, this will be the targetPackage specified in the <javaModelGenerator> configuration, plus the sub-package if sub-packages are enabled, plus the calculated domain object name.
