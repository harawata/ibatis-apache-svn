The <abatorConfiguration> Element

The <abatorConfiguration> element is the root element of an Abator configuration file. The file should contain the following DOCTYPE: 

<!DOCTYPE abatorConfiguration PUBLIC
  "-//Apache Software Foundation//DTD Abator for iBATIS Configuration 1.0//EN"
  "http://ibatis.apache.org/dtd/abator-config_1_0.dtd">

Required Attributes

None

Optional Attributes

None

Child Elements

  • <properties> (0 or 1)
  • <abatorContext> (1..N)
