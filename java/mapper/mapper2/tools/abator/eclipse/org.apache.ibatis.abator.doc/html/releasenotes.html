Abator Eclipse Feature Release Notes

Version 0.7.0

  • Upgraded to Abator Version 0.7.0
  • Upgraded to Eclipse Version 3.1
  • The Java file merger now fully supports JSE 5.0
  • Fixed bug where multi-segment source paths failed
  • Use an adaptable type to associate Abator configuration files with the popup menu items instead of content type. This should resolve conflicts with other plugins like oXygen

Version 0.6.5

  • Upgraded to Abator Version 0.6.5
  • Changed EclipseShellCallback so that different source folders can be specified

Version 0.6.2

  • Made our content type high priority to avoid conflicts with other XML enabled plugins

Version 0.6.1

  • Upgraded to Abator Version 0.6.1
  • Added an Eclipse enabled Ant task
