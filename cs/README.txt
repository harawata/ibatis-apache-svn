DevGuide - The docbook source for the Developers Guide. Form this, we generate HTML, PDF, and CHM files to be distributed in the Documentation package.

Documentation - Generated HTML, PDF, and CHM files to post to the website. Nothing here is "original". It is all output generated from source files in other packages. 

Forrest - We generate the website infrastructure with Apache Forrest, using the source files kept here. The site output is checked in under the Documentation package. 

Source1 - The main iBatisNet solution, including Common, DataAccess, and DataMapper.

Tutorial - The tutorial solution (and docbook).

NPetstore - The example PetShop solution

Whiteboard - Proposals and other internal team documentation.
